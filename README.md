**第一章 · 交易框架与事件驱动**

**1.JingniTrader**

**一个基于Python的量化交易开发框架**

**学习目标**：理解JingniTrader的设计理念、整体结构以及如何通过配置适应不同券商量化交易软件的策略环境。

**核心内容**：详解券商三款主流量化交易软件(ptrade/qmt/goldminer)的设置，阐述框架如何通过条件判断实现一套代码兼容运行。
